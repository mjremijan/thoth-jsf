Hello World! This is a basic JSF xhtml page!

The source for this page is index.xhtml.

The web.xml file for this application maps FacesServlet to the *.xhtml extension. It is important to have the *.xhtml extension mapped so if a user puts index.xhtml into the browser, the user won't get the source code of the XHTML page.
